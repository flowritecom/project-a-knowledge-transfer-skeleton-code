## Prerequisites
For secret fetching, insert aws api keys to .aws_creds

### for macOS - pyenv can be useful
`brew install pyenv`
`brew install openssl readline sqlite3 xz zlib`

### for dev or actions with poetry, you might need python locally
`pyenv install 3.9.9`
`pyenv local 3.9.9`
`poetry install`
